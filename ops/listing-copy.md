# 发布文案

## GitHub 项目简介

RiskLens CN：中文内容发布前的风险扫描与改写建议工具。适合副业项目、商品详情页、开源项目 README、小红书/闲鱼文案和私域客服话术。免费版纯前端运行，无需登录，不上传用户数据。

## 小红书/朋友圈短文案

标题：

```text
我做了一个中文文案发布前风险扫描工具
```

正文：

```text
很多副业文案不是产品不好，而是表达太容易踩坑：保证收益、零风险、100%准确、批量获客、代写包过、官方合作。

我做了一个免费小工具 RiskLens CN，把文案粘进去，会提示风险等级、命中原因和改写方向。适合发布商品详情、开源项目 README、闲鱼服务、小红书笔记前自查。

免费版已开源，Pro 版会加入行业规则包和报告模板。
```
