# RiskLens CN

RiskLens CN 是一个中文内容风险扫描与改写建议工具。它面向个人副业、商品详情页、开源项目 README、小红书/闲鱼发布文案和私域客服话术，帮助发布前发现常见合规、平台和信任风险。

当前仓库是免费版 MVP：纯前端静态页面，无需后端、无需数据库、无需登录。

## 功能

- 输入主题、标题或完整文案
- 选择使用场景：通用、销售、副业收益、健康、教育、开源项目
- 自动识别高频风险词和风险类型
- 输出风险等级、命中证据、改写建议和发布前清单
- 支持直接部署到 GitHub Pages、Vercel、Netlify 或任意静态网站服务

## 合规边界

RiskLens CN 只做信息整理、规则提示和表达优化，不提供法律、税务、医疗、金融投资或平台审核结论。工具输出需要人工复核，不能替代律师、医生、持牌金融机构或平台官方规则。

本项目不提供刷量、规避审核、代写造假、爬取隐私数据、黑灰产引流等能力。

## 本地运行

直接用浏览器打开 `index.html` 即可。

也可以启动一个本地静态服务器：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```
