# RiskLens CN Pro 路线

## 产品定位

输入一个主题、项目说明或推广文案后，系统自动从三个层面做风险把控：

1. 文案层：绝对化宣传、收益承诺、隐私数据、医疗健康、教育诚信、知识产权、平台风控。
2. 公开信息层：搜索相关监管提醒、平台规则、投诉案例、竞品争议、许可证要求。
3. 改造层：输出可发布版本、风险声明、售后边界、免责声明和发布前清单。

## 合法联网搜索方案

不要做未授权爬虫或批量采集个人信息。推荐使用：

- 官方搜索 API 或合规第三方搜索 API
- 平台公开规则页面
- 政府/监管机构公开公告
- 开源许可证数据库
- 用户主动提供的网页链接

联网结果只做摘要和风险提示，不复制大段版权内容，不绕过登录墙、付费墙或反爬限制。

## Pro 功能清单

- `searchRisk(topic)`：搜索公开网页并提取风险线索
- `sourceMap`：记录来源标题、链接、发布时间和摘要
- `industryRules`：行业规则包，例如副业项目、教育服务、健康产品、招商加盟、AI 工具
- `rewriteModes`：保守版、平台发布版、客服解释版、详情页版
- `exportReport()`：导出 PDF 或 Markdown 风险报告
- `teamKeywords`：用户自定义敏感词库

## 技术架构建议

免费版：

```text
HTML + CSS + JS + 本地规则库
```

Pro 版：

```text
前端静态页面
  -> 后端 API
  -> 搜索 API / 官方规则源
  -> 风险分类器
  -> 报告导出
```

可选部署：

- Vercel Functions
- Cloudflare Workers
- Supabase Edge Functions

## 变现路径

GitHub 免费版负责曝光和信任背书。付费版不要放在公开仓库。

推荐链路：

```text
GitHub 免费项目
  -> README 展示案例
  -> 购买 Pro 规则库/报告模板
  -> 付款后交付 zip 包或私有仓库邀请
```
